# Pasta de Uploads da Galeria

Esta pasta armazena as imagens enviadas através do painel administrativo.

## Como adicionar fotos:

1. **Via Upload (Recomendado):**
   - Vá em Admin → Gallery
   - Clique em "Add Image"
   - Clique em "Enviar Foto"
   - Selecione a imagem do seu computador
   - Aguarde o upload

2. **Manualmente:**
   - Copie suas imagens para esta pasta
   - Use nomes como: style1.jpg, style2.jpg, etc.
   - No painel admin, use a URL: `/gallery-uploads/nome-do-arquivo.jpg`

## Formatos Suportados:
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)
- GIF (.gif)

## Tamanho Máximo:
- 5MB por arquivo

## Exemplo de URLs:
- `/gallery-uploads/style1.jpg`
- `/gallery-uploads/meu-look.jpg`
- `/gallery-uploads/foto-cliente.png`
